# ObaX Innovations Limited

Static website for ObaX Innovations Limited, an innovation company building marketplace platforms, software, applications, websites, apps, and data visualization products for Nigeria and Africa.

## Highlights

- High-tech visual identity and generated hero asset
- Responsive landing page for desktop and mobile
- Showcase sections for FixAm9ja, Nigeria Election Dashboard, Inventory Software, and custom builds
- Static HTML/CSS/JavaScript with no build step required

Open `index.html` locally or publish the repository with GitHub Pages.
